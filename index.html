作者選股方式 :



1.首先檢查 


 - 產業 (民生產業 非景氣循環股 有寡佔獨佔性質或產業前兩名 具備護城河條件者 優先考慮) 

 - 稅後盈餘     (獲利)  穩定度 

 - 股價             (本金)  穩定度 

 - 現金殖利率 (利息)  穩定度


2.接著檢查


( 營收 / 毛利 / 營業利益 / 業外損益 / 稅後淨利 )


( EPS / PER / PBR / ROE / ROA )


( 負債比 / 流動比 / 速動比 / 利息保障倍數 )


( 應收帳款 / 應付帳款 / 存貨 / 固定資產 / 總資產 之 周轉率)


( 營業現金流量 / 自由現金流量 / 淨現金流量 )



3.其他檢查


- 籌碼面: 董監持股 / 三大法人持股 / 持股等級比率


- 消息面: CMoney / MoneyDJ / 財報狗 / PTT


- 研究年報
